Basic Usage

DownJS is built with two main things in mind: Be Fast and Be Easy. To accomplish that, we've simplified the DownJS process into three easy steps:

  1. Load the DownJS Script
  2. Request the scripts you would like to load
  3. Do whatever you want in the callback

Load the DownJS Script

The DownJS Script is where all the magic of loading the javascript files happens. Simply load it by calling http://script.downjs.com like so:

You do not need to specify any file after the script.downjs.com domain.

<script type="text/javascript" src="http://script.downjs.com"></script>

Request the scripts you would like to load

The downjs function accepts two parameters: a scripts array and a callback function. The scripts array is simply an array of strings containing the names of the javascript files you would like to load. For instance, if you wanted to load jQuery and Raphael you would do this:

The scripts array expects the name of the javascript plugin, not the name of the file.

downjs(['jquery', 'raphael'], /*Optional Callback*/);

Do whatever you want in the callback

We keep track of how the scripts are loading, and provide a callback option so we can keep you in the loop too.

The callback function is optional

downjs(['jquery', 'raphael'], function() {
	alert("All scripts  are loaded!");
});
